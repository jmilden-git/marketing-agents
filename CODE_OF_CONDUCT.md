# Code of Conduct

We follow the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/). By participating, you agree to:

- Be respectful and inclusive.
- Give and accept constructive feedback.
- Assume good intent, and ask for clarification when needed.
- Report unacceptable behaviour to the maintainers.

For incidents, contact the maintainers with context and evidence so we can respond promptly.
